# Christoph's Dotfiles

These are my dotfiles. There are many like it, but these ones are mine.

## Requirements

 * [Legit][]
 * ZSH
 * Rake (for installing)

## Install

    git clone git://github.com/CHH/dotfiles ~/.dotfiles
    cd ~/.dotfiles
    rake install

This links all files ending with `.symlink` into the home directory.
